## Methods

### Nearest Neighbour

Parameters: 

- Start city
### Local Search

Parameters: 

- Operation: Swap, Insert, Invert
- Iterations: 5-25

### Genetic Algorithm

Parameters: 

- Population size: 10-100
- Generations: 10-100
- Selection: Rank
- Crossover: OX
- Mutation: Swap, Insert, Invert
- Mutation rate: 1-10 %
